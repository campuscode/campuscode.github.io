Muitas pessoas que se interessam pelo desenvolvimento em Ruby e Rails e utilizam Windows como SO enfrentam dificuldades para configurar o ambiente. Na verdade conheço pessoas que desistiram já na instalação :(

Confesso que não sou fã do Windows como SO para desenvolvimento. Porém sei que muitas pessoas gostam desse ambiente e não acho justo restringir o uso de um linguagem desta forma. A falta de tutoriais atualizados sobre este assunto motivou escrever este post.

Numa rápida pesquisa pelo Google aparecem algumas opções que prefiro descartar. Destaco algumas:

  • Rails Installer</a>: A proposta é muito interessante porém está defasada. A última versão oferece somente Ruby 1.9 e Rails 3.2.</li>
  • Nitrous.io</a>: Outra idéia muito interessante e poderosa. Uma plataforma na nuvem pronta para o desenvolvimento em diversos frameworks e que permite o uso de editores locais. O ponto negativo? É pago. Vamos lá, não queremos gastar dinheiro para testar algo (pelo menos não por enquanto).</li>
    </ul>
    Voltando ao que interessa, aqui no blog vamos apresentar duas formas de instalar todo ambiente Ruby on Rails no Windows. Começando com a mais trabalhosa mas que permite entender um pouco melhor como toda mágica começa a acontecer.

    Ruby Installer + Prompt de Comando</strong>

    Acesse o site do RubyInstaller</a> e faça o download da versão mais recente.

    [caption id="attachment_11" align="aligncenter" width="449"]</a> Site do RubyInstaller[/caption]

    Com o download concluído, execute o instalador. Não esqueça de habilitar a opção</strong> 'Add Ruby executables to your PATH'.

     

    [caption id="attachment_13" align="aligncenter" width="506"]</a> Instalador do Ruby Installer[/caption]

     

    Pronto! Com a instalação concluída você já pode acessar o Prompt de Comando</a> ou o Windows PowerShell e executar o comando:

    ruby -v</em></strong></p>
         </a>

    O retorno esperado, como visto acima, é a versão instalado dor Ruby em sua máquina.

    Em seguida faça download do Development Kit,</strong> também disponível na página de downloads</a> do Ruby Installer. No caso da versão do Ruby instalada enquanto escrevo esse tutorial, o download correto seria Ruby 2.0/2.1 32bits. Fique atento a versão do Ruby e a arquitetura do seu Windows ao escolher a opção de download.

     

    [caption id="attachment_15" align="aligncenter" width="471"]</a> Opções de download para o Development Kit[/caption]

    Após concluir o download execute o arquivo e ao escolher a pasta destino procure a pasta de instalação do Ruby (geralmente C:\Ruby(numero_de_versão) e acrescente '\devkit'.

    [caption id="attachment_16" align="aligncenter" width="350"]</a> Pasta destino do DevKit[/caption]

    Para instalar o Development Kit, volte ao terminal e acesse a pasta informada no passo anterior. Em seguida execute os comandos :

    ruby dk.rb init</strong></em></p>

    ruby dk.rb install</strong></em></p>

    [caption id="attachment_17" align="aligncenter" width="679"]</a> Instalação do Development Kit no Prompt de Comando[/caption]

     

    Para finalizar vamos instalar o Ruby on Rails</a>. Um pré-requisito para essa instalação é possuir algum banco de dados já instalado. Neste tutorial vamos utilizar o SQLite3</a>.

    Acesse a página de downloads</a> do SQLite.org. Procure pela seção Precompiled Binaries for Windows </b>e faça download dos arquivos sqlite-shell-win32-x86*.zip e sqlite-dll-win32-x86*.zip.  </b>Crie a pasta C:\sqlite3 </strong>e descompacte os 2 arquivos lá dentro.

    Em seguida, volte ao Prompt de Comando e execute o seguinte comando:

    setx path "%path%;c:\sqlite3"</strong></em></p>

    Pronto! O SQLite deve estar disponível para uso. Para verificar execute o comando sqlite3 --version </strong></em> no Prompt.</p>

    [caption id="attachment_19" align="aligncenter" width="672"]</a> Verificando a instalacao do SQLite3[/caption]

    A última parte do tutorial é justamente a mais simples. A instalação do Ruby on Rails é feita pelo Prompt de Comando usando a ferramenta RubyGems</a> que já está disponível graças a instalação do Ruby que realizamos.

    Basta acessar o Prompt e executar o comando gem install rails</strong></em>. Várias bibliotecas auxiliares ao Rails serão instaladas automaticamente.

    [caption id="attachment_20" align="aligncenter" width="672"]</a> Instalação do Rails em andamento...[/caption]

    Após a instalação você pode conferir a versão do Rails instalada executando o comando rails -v </strong></em>no Prompt.

    Pronto!</strong> Seu Windows está pronto para começar seu primeiro projeto Ruby on Rails :D
