Alguns dias atrás publiquei aqui um tutorial para instalar Ruby e Rails no Windows via Prompt de Comando. </a> Foi um post bem legal de se escrever!

Acredito que a dificuldade daquele tutorial é baixa. São apenas alguns downloads e meia dúzia de comandos no Prompt. Mas entendo que sempre tem alguém que quer colocar logo a mão no código sem se preocupar com instalações e configurações.

Foi pensando nesse grupo que Bryan Bibat</a> criou o projeto Rails FTW</a>.

[caption id="attachment_26" align="aligncenter" width="599"]</a> Página inicial do projeto Rails FTW[/caption]

 

Trata-se de um instalador no melhor estilo Windows: Next, Next, Finish. Então é so fazer o download e executar.Novamente temos somente um ponto de atenção</strong>: deixe marcada a opção 'Add Ruby executables to your PATH'.

 

[caption id="attachment_27" align="aligncenter" width="504" class=" "]</a> Instalação do Rails FTW[/caption]

Ao fim da instalação você poderá acessar o Rails FTW via menu Iniciar.

[caption id="attachment_31" align="aligncenter" width="411"]</a> Rails FTW no menu Iniciar[/caption]

Escolha a opção Start Command Prompt with Ruby</strong> e pronto! Para conferir as instalações você pode usar os comandos ruby -v </strong></em>e rails -v</strong></em>.

[caption id="attachment_32" align="aligncenter" width="666"]</a> Conferindo instalação do Rails FTW[/caption]

É isso! Espero que tenha ajudado mais uma vez!
