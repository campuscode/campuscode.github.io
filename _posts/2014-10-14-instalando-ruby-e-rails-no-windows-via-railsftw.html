Alguns dias atrás publiquei aqui um tutorial para instalar Ruby e Rails no Windows via Prompt de Comando.  Foi um post bem legal de se escrever!

Acredito que a dificuldade daquele tutorial é baixa. São apenas alguns downloads e meia dúzia de comandos no Prompt. Mas entendo que sempre tem alguém que quer colocar logo a mão no código sem se preocupar com instalações e configurações.

Foi pensando nesse grupo que Bryan Bibat criou o projeto Rails FTW.

 

Trata-se de um instalador no melhor estilo Windows: Next, Next, Finish. Então é so fazer o download e executar.Novamente temos somente um ponto de atenção: deixe marcada a opção 'Add Ruby executables to your PATH'.

 

Ao fim da instalação você poderá acessar o Rails FTW via menu Iniciar.

Escolha a opção Start Command Prompt with Ruby e pronto! Para conferir as instalações você pode usar os comandos ruby -v e rails -v.

É isso! Espero que tenha ajudado mais uma vez!
